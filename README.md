# Employee Management System
Java Swing application for managing employee information with MySQL database.  
Supports: Add, Update, Delete, Search, Print Employee List, Payroll Management.

##  Features
- Dashboard with background image
- Add new employee
- View/Update/Delete/Print employee list
- Payroll management

<h2 style="border: none; margin-bottom: 0;"><b>How to Run</b></h2>

- Setup Database: Install MySQL Server and Workbench, then run the database.sql script (located in the src folder) to create the employee_management database with sample data.
- Configure Project: Open the project in IntelliJ IDEA and add the MySQL Connector/J library (located in the lib folder or downloaded online).
- Update Credentials: Change the database login info in util/DatabaseConnection.java if needed (default: user=root, password=empty).
- Execution: Run the Main.java file to start the application.
